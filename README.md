- 純 HTML5 / CSS3 / JavaScript，無框架
- Google Fonts：Playfair Display · Noto Serif TC · DM Sans
- YouTube iframe 嵌入（音樂卡片）
- Intersection Observer scroll reveal 動畫
- RWD 響應式設計（768px 斷點）
- 色彩系統：Cream / Ink / Terracotta / Sage / Brown / Gold
